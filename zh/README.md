# pixcanvas.fun

[English](README.md) | [Français](docs/fr/README.md) | [Deutsch](docs/de/README.md) | [العربية](docs/ar/README.md) | [中文](docs/zh/README.md) | [हिन्दी](docs/hi/README.md) | [Српски](docs/sr/README.md) | [Español](docs/es/README.md) | [Italiano](docs/it/README.md)

欢迎来到 **pixcanvas.fun**，一个合作与创造像素艺术的游戏！通过 **pixcanvas.fun**，您可以与他人互动，释放您的创造力，并在共享画布上创建惊人的像素艺术。

## 主要功能

- **协作画布**：与他人实时共享一个大型像素画布。
- **用户友好的界面**：使用多样化的调色板和简单的工具进行绘画。
- **修改历史**：跟踪贡献并在需要时恢复更改。
- **账户系统**：保存您的创作并访问定制功能。

## 使用的技术

- **React**
- **JavaScript (JS)**
- **TypeScript (TS)**
- **CSS**
- **JSON**

## 本地安装与运行

请按照以下步骤设置项目并在本地运行：

1. **克隆仓库**：

   ```bash
   git clone https://github.com/your-username/pixcanvas.fun.git
   ```

2. **安装所需依赖**：

   ```bash
   cd pixcanvas.fun
   npm install
   ```

3. **启动开发服务器**：

   ```bash
   npm start
   ```

4. **访问应用**：
   打开浏览器并访问 `http://localhost:3000`。
